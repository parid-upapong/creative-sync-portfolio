# Recommended Tech Stack for Implementation

## Frontend
- **Framework:** Next.js 14 (App Router) for SEO and performance.
- **Styling:** Tailwind CSS + Framer Motion (for "Magic" feel).
- **State Management:** Zustand (lightweight for editor state).

## Backend & AI
- **Language:** Python (FastAPI) for AI processing.
- **AI Models:** 
  - Vision: GPT-4o or Claude 3.5 Sonnet (Image analysis & Copywriting).
  - Image Processing: Sharp (Node.js) or Pillow (Python).
- **Database:** PostgreSQL (Supabase) for user data and metadata.

## Infrastructure
- **Storage:** AWS S3 or Google Cloud Storage (for raw assets).
- **Deployment:** Vercel (Frontend) & Railway/Render (AI Worker).
- **CDN:** Cloudflare (for high-speed portfolio delivery).
