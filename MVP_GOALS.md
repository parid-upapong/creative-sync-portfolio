# MVP (Minimum Viable Product) Objectives

## Objective 1: Smart Asset Ingestion
- [ ] Support for Bulk Image/Video Upload.
- [ ] Integration with Google Drive/Dropbox.
- [ ] AI Metadata Extraction: Automatically generate titles, tags, and descriptions from images.

## Objective 2: The "Magic" Generator (V1)
- [ ] 3 Core Design Styles: Minimalist, Bold, and Brutalist.
- [ ] AI Color Palette Extraction: Generating site themes based on the dominant colors in the portfolio pieces.
- [ ] Instant Mobile Optimization.

## Objective 3: Seamless Deployment
- [ ] Subdomain hosting (username.platform.ai).
- [ ] SEO Meta-tag auto-generation.
- [ ] Basic Analytics Dashboard.

## Objective 4: User Experience
- [ ] Interactive "Chat-to-Edit" interface (e.g., "Make the header more aggressive" or "Use a serif font").
